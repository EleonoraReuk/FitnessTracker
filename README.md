# FitnessTracker# Fitness Tracker

## Описание проекта

Fitness Tracker — приложение для отслеживания тренировок и достижения спортивных целей. Пользователь может создавать цели, фиксировать выполненные тренировки и анализировать свой прогресс.

Основная задача приложения — помочь пользователям контролировать процесс достижения спортивных результатов и своевременно получать уведомления о важных изменениях прогресса.

---

## Постановка задачи

Многие люди начинают заниматься спортом, но теряют мотивацию из-за отсутствия контроля над результатами. Необходимо разработать приложение, которое позволит:

- создавать спортивные цели;
- отслеживать выполненные тренировки;
- хранить историю активности;
- отображать прогресс достижения целей;
- уведомлять пользователя о важных событиях.

### Примеры целей

- Пробежать 100 км за месяц;
- Выполнить 50 тренировок;
- Сбросить 5 кг;
- Посещать спортзал 3 раза в неделю.

---

## Функциональные возможности

### Работа с целями

Пользователь может:

- создавать новую цель;
- просматривать список целей;
- изменять параметры цели;
- удалять цель.

### Работа с тренировками

Пользователь может:

- добавлять тренировку;
- указывать дату тренировки;
- указывать длительность;
- указывать тип активности.

### Анализ прогресса

Приложение автоматически:

- подсчитывает общий прогресс;
- определяет процент выполнения цели;
- показывает текущую статистику.

### Система уведомлений

Пользователь получает уведомления при:

- достижении 25% цели;
- достижении 50% цели;
- достижении 75% цели;
- полном выполнении цели.

---

## Используемый паттерн

### Observer (Наблюдатель)

Для реализации системы уведомлений используется паттерн Observer.

Паттерн позволяет автоматически уведомлять пользователя об изменении прогресса достижения цели. Благодаря этому логика целей не зависит от способа отображения уведомлений.

Основные компоненты паттерна:

- **Goal** — объект, за которым ведётся наблюдение;

- **IObserver** — интерфейс для получения уведомлений;

- **UserNotification** — класс, который получает уведомления и отображает сообщения пользователю.

### Пример работы

Цель:

```text

Пробежать 100 км

```

После добавления тренировки:

```text

Текущий результат: 25 км
Прогресс: 25%

```

Уведомление:

```text

Поздравляем! Выполнено 25% цели.

```
